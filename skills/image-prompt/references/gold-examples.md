# Gold examples — few-shot quality anchors

These are the reference prompts the skill emulates for **richness and style**. Read the
ones matching the target archetype before writing, to calibrate depth and phrasing.

**This file is meant to grow** — add favourites over time. Format: `## N. Title — [archetype]`
then a one-line "Teaches:" tag, then the prompt. Keep them faithful; don't water them down.

Coverage gaps to fill later (styles not yet represented): pure typographic/Swiss poster;
clean 3D product render / packaging; real SaaS dashboard UI; brand mascot / character sheet;
album cover; isometric / Blender illustration.

## From the gpt-image-2 official guide (structured/brief style)

Shorter, brief-style prompts also work well for these archetypes:
- **Infographic:** name the topic + audience + "understand technically and visually the flow"; quality high.
- **Photoreal candid:** "photorealistic candid photograph of…", 35mm/50mm, shallow DoF, real skin texture, "honest and unposed, no glamorization".
- **Logo:** "original, non-infringing logo for [brand]", clean vector shapes, strong silhouette, balanced negative space, scalable, flat, plain background, no watermark.
- **Ad:** written as a creative brief — brand + audience + concept + composition + exact tagline in quotes rendered once, "no extra text, no watermarks, no unrelated logos".
- **UI mockup:** describe the product as if it already exists — layout, hierarchy, real components, "looks like a real, well-designed app", in a device frame.
